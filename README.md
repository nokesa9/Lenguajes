## Ejecutar el codigo

* Descargar el proyecto y descomprimir el archivo
* Ingresar a la carpeta del proyecto
    i. bison -d p1.l
    ii. flex p1.l
    iii. gcc p1.tab.c lex.yy.c -o calculadoraBison -lfl
    iv. ./calculadoraBison


En el archivo explicacion.pdf se encuentra la solucion a las preguntas.
